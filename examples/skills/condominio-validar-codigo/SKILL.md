# Condo Code Validation

Use this skill to review the implementation associated with the issue.

Check:

1. Adherence to the acceptance criteria.
2. Bugs, regressions, and unexpected behavior.
3. Test coverage proportional to risk.
4. Readability and maintainability.
5. Automated check results.

When you find small actionable problems, fix them. When you find product risk or
a product decision, comment the blocker and add `blocked:human-help`.
