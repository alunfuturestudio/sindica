# Condo Refine

Use this skill to turn an open issue into a clear request.

Produce:

1. Summary of the problem or opportunity.
2. Objective questions when there is ambiguity.
3. Initial acceptance criteria.
4. Risks, dependencies, and points that require a human decision.

Do not implement code in this phase. If there is a blocker, comment the blocker
and add `blocked:human-help`.
