# Condo PRD

Use this skill to write a short, implementable, and testable PRD.

Include:

1. Context and goal.
2. In-scope and out-of-scope items.
3. Acceptance criteria.
4. High-level technical plan.
5. Expected tests.
6. Risks and pending decisions.

Do not implement code in this phase.
