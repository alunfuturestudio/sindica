# Condo Approve

Use this skill after human homologation has been approved.

Check:

1. The PR is up to date and reviewable.
2. Relevant checks are passing or have a clear justification.
3. Acceptance criteria are covered.
4. Final comment includes a summary of what was delivered.

If anything blocks final approval, comment the blocker and add
`blocked:human-help`.
